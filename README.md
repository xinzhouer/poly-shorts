# Poly Shorts MVP

An immersive short video drama application built with React, Vite, and Tailwind CSS.

## Features
- **Home**: Browse featured and trending drama series.
- **Detail**: View series information and episode list.
- **Player**: Immersive vertical video player with swipe-like navigation.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand (State Management)
- React Router DOM

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure
- `src/components`: Reusable UI components.
- `src/pages`: Page components (Home, Detail, Player).
- `src/data`: Mock data and data access helpers.
- `src/store`: Global state management.
- `src/types`: TypeScript interfaces.
